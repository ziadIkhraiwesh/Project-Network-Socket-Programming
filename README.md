# ENCS3320 – Project 1 (Socket Programming)

##  Description

This project implements:

* A simple HTTP web server using Python sockets (Task 2)
* A hybrid TCP/UDP multiplayer guessing game (Task 3)

---

##  Structure

* **Task2/**: Web server + HTML/CSS + images
* **Task3/**: Server and client for the game
* **Report/**: Full project documentation (PDF)

---

##  How to Run

### Task 2 (Web Server)

```bash
python Task2/server.py
```

Open:

```text
http://localhost:9971
```

### Task 3 (Game)

```bash
python Task3/server.py
```

In another terminal:

```bash
python Task3/client.py
```
